# Examples

Runnable `matterbridge` examples. Each script is self-contained — run it with
`python examples/<name>.py`.

- [light.py](./light.py) — Basic On/Off, Dimmable, and Extended Color Light implementation.
- [plugpower.py](./plugpower.py) — Smart Plug with electrical energy and power measurement.
- [sensors.py](./sensors.py) — Multi-sensor suite (Temperature, Humidity, Pressure, Light, Occupancy, Contact).
- [fan.py](./fan.py) — Fan Control with multi-speed support and mode sequencing.
- [thermostat.py](./thermostat.py) — Thermostat with setpoints and local temperature updates.
- [curtain.py](./curtain.py) — Window Covering (Curtain) control with position management.
- [doorlock.py](./doorlock.py) — Door Lock with security command handling.
